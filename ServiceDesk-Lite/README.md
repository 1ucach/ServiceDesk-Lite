# ServiceDesk Lite

Учебный проект для производственной практики ПП.02.

## Назначение

ServiceDesk Lite — информационная система для учета клиентов и заявок предприятия.

## Возможности

- просмотр главной страницы;
- учет клиентов;
- учет заявок;
- изменение статусов заявок;
- хранение истории взаимодействий;
- тестирование с помощью pytest.

## Запуск

```bash
pip install -r requirements.txt
python run.py
```

После запуска открыть:

```text
http://127.0.0.1:5000
```

## Тестирование

```bash
pytest
```
