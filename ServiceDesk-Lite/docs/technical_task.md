# Техническое задание

## Наименование

Информационная система учета клиентов и заявок ServiceDesk Lite.

## Цель разработки

Автоматизация процессов обработки клиентских заявок.

## Требования

- хранение клиентов;
- регистрация заявок;
- изменение статусов;
- просмотр информации;
- тестирование основных функций.

## Среда функционирования

- Windows 10;
- Python 3;
- SQLite;
- веб-браузер.
