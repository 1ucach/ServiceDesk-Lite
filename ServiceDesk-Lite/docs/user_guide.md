# Руководство пользователя

## Назначение системы

ServiceDesk Lite предназначена для учета клиентов и обработки заявок.

## Запуск

1. Установить зависимости.
2. Запустить `python run.py`.
3. Открыть `http://127.0.0.1:5000`.

## Основные функции

- просмотр главной страницы;
- добавление клиентов;
- создание заявок;
- просмотр списка заявок;
- просмотр карточки заявки.
